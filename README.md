## learning_log
### 使用Django-1.10 做的一个学习笔记的记录网站
#### 需要安装
* `django-bootstrap3`
* `Django < 2.0`
#### 运行: `python manage.py runserver 127.0.0.1:8001`
#### 浏览器打开 127.0.0.1:8001
#### 管理员账号是默认的
